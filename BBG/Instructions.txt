1) Если есть файл с именами файлов, присланный НАСА (У меня, например, это файл allTracks16.txt) , его надо
с помощью getCorrectNames.m перевести в нужный формат имен: указать имя файла, который нужно преобразовать, и в какой.
Запустить, теперь у нас есть файл с нормальными именами ( у меня allTracks_16.txt). 

2) Далее надо сделать comfile, для этого запускаем CreateComfile.m, там все подписано, в первый   - только что сделаный
 файл с именами треков, во второй имя для comfile"а. Далее внутри цикла есть переменная path, ее нужно поменять на путь к корневой
папке с файлами для обработки. Запустить, теперь у нас должен быть нормальный comfile ( у меня comfile16_h1)

3)В программе Copy_of_Execute.m надо поменять в начале имя comfile'а, на тот, что у вас.
